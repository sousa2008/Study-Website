# 🌐 Projeto Web (HTML, CSS e JavaScript)

Projeto desenvolvido para praticar conceitos de desenvolvimento web que aprendo por conta própria e na faculdade.

## 🚀 Sobre o projeto
Este projeto consiste em uma página web criada utilizando HTML e estilizada com CSS.
Ele simula um site de estudos e,a cada versão, ficamais funcional

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 📚 Aprendizados
- Estruturação de páginas com HTML
- Estilização com CSS
- Organização de arquivos
- Noções iniciais de desenvolvimento web

## ▶️ Como visualizar
Abra o arquivo `index.html` no navegador.
